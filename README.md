## FIT5046 Pain Diary

### Description

An Android Kotlin Project which is to record the pains. 

Authication: Google Auth

Database: Google Firebase

Weather API: Openweathermap

Map API: Mapbox

### Tip

Before running, change API key in *strings.xml* and *HomeFragment.kt*
